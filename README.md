# FIFA World Cup 2026 Analysis

Dashboard analisis visual yang memetakan kekuatan negara-negara kontestan FIFA World Cup 2026. Proyek ini mengintegrasikan data nilai pasar (market value) skuad dengan performa produktivitas (G+A) pemain kunci di level klub pada musim 2025/2026 untuk memprediksi potensi progresi negara di turnamen.

## Demo
https://[masukkan-link-vercel-kamu-disini].vercel.app

## Dashboard
- **Kedalaman Skuad (Market Value)**: Bar chart yang membandingkan total nilai pasar skuad (€ Miliar) antar negara untuk mengukur kekuatan finansial dan kualitas kedalaman skuad.
- **Firepower Matrix**: Scatter plot yang mengkorelasikan total kontribusi Gol+Assist pemain kunci di level klub dengan nilai pasar negara sebagai indikator performa produktivitas.
- **Prediksi Babak Terjauh**: Bar chart horizontal yang menampilkan proyeksi capaian babak maksimal yang bisa diraih negara berdasarkan data performa kolektif pemain pilar.
- **Tabel Pemain Kunci**: Daftar referensi 3 pemain dengan nilai pasar tertinggi per negara untuk transparansi data dan konteks taktis di setiap posisi.
